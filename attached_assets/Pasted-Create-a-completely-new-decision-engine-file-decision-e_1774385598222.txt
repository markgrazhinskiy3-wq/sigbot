## Architecture

One file, under 300 lines. Two strategies: Level Bounce + Level Break.

## Step 1: Calculate indicators on 1M data

RSI(14) on 1M candles
MACD(12, 26, 9) on 1M candles — we need histogram values
CCI(20) on 1M candles
ATR(14) on 1M candles
EMA(20) on 1M candles

Calculate these manually using numpy/pandas. Do NOT import TA-Lib or any external TA library. Simple formulas only.

## Step 2: Build support/resistance levels from 1M candles

Scan all available 1M candles (typically 30-50)
Find price zones where highs/lows cluster
Level = zone where price touched 2+ times
Strong level = 3+ touches
Touch = high or low within 0.05% of level price
Track number of touches for each level

## Step 3: Strategy 1 — Level Bounce

BUY signal (bounce from support):
  1. Current price within 0.08% of a support level
  2. MACD histogram: current bar > previous bar (momentum shifting up, slowing decline)
  3. Current candle shows bounce: lower_shadow > body_size
  4. RSI(14) is NOT above 70 (not overbought)
  5. CCI(20) < 50 (room to move up)

SELL signal (bounce from resistance):
  1. Current price within 0.08% of resistance level
  2. MACD histogram: current bar < previous bar (momentum shifting down)
  3. Current candle: upper_shadow > body_size
  4. RSI(14) is NOT below 30
  5. CCI(20) > -50 (room to move down)

## Step 4: Strategy 2 — Level Break

BUY signal (break above resistance):
  1. Resistance level with 3+ touches (strong level)
  2. Current candle CLOSED above the level
  3. Current candle body > average body of last 10 candles (strong impulse)
  4. MACD histogram growing for last 2+ bars (increasing momentum behind the break)
  5. RSI between 45-75 (has momentum but not exhausted)
  6. CCI > 0 (confirms bullish)

SELL signal (break below support):
  1. Support level with 3+ touches
  2. Current candle closed below level
  3. Body > average body
  4. MACD histogram declining 2+ bars
  5. RSI between 25-55
  6. CCI < 0

## Step 5: Filters (only 2)

1. Dead market: ATR(14) < 0.001% of current price → NO signal
2. Conflict: if bounce says BUY and break says SELL simultaneously → NO signal

## Step 6: Signal priority and stars

If both strategies trigger same direction → 5 stars (strongest)
If only bounce triggers → 4 stars
If only break triggers → 3 stars
